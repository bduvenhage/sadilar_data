xhosa.2846.food safety
ABANTU bayathanda ukuzonwabisa neentsapho nabahlobo babo ehlotyeni, ukanti nokutya kunendima enkulu kwezo ndibano. Ngelishwa kungenzeka ukutya okuhle kubize neentsholongwane ebezingalindelekanga.
Uguliswa kukutya okuneentsholongwane ezanda ngokukhawuleza zize zibangele ityhefu. Ezo tyhefu ke zixabana nezisu zethu, size sigule.
Ungaqala ukugula emva kweeyure ezingama-48 ukutyile ukutya okungalunganga. Kuxhomekeka ekubeni ukutya bekutyhefeke kangakanani kodwa iimpawu ziquka isicaphu-caphu, ukuhlanza, ukurhuda nokulunywa sisisu.
Iintsana, abantu abalupheleyo, nabo banamajoni omzimba abuthathaka ngabona baguliswa kakhulu kukutya okonakeleyo kuba baphelelwa ngamanzi emzimbeni balahlekelwe nazizakha-mzimba ezibalulekileyo.
Bona ke bangafa nokufa kukutya ukutya okonakeleyo, ngoko kubalulekile ukuba bafumane unyango olukhawulezileyo.
Amaxesha amaninzi kukungaphathwa kokutya ngendlela okwenza ukutya kutyhefeke, kodwa xa sikuphatha ngononophelo singakuthintela ukosuleleka kweentsapho zethu.
Emasikwenze ukuze kungonakali ukutya
n Hlambisisa izandla ngaphambi kokuphatha ukutya. (Oko kuthintela ukusasazeka kweentsholongwane.)
n Qinisekisa ukuba wonk' umntu uhlamba izandla emva kokuya ngasese, emva kokuphatha izilwanyana zasekhaya, emva kokusebenza esitiyeni nangaphambi kokutya.
n Hlamb' izandla xa ubuphethe inyama okanye intlanzi ekrwada. Zihlambe ngamanzi nesepha imizuzwana engama-20.)
n Coca iindawo zokulungisela ukutya zasekhitshini ngamanzi ashushu anesepha okanye ugalele nentwana yebhlitshi. Iitreyi nezithebe zokunqunqela mazicoceke yaye zihlanjwe emva kokusetyenziswa.
n Hlamba izitya ngamanzi ashushu. Hlamba iisinki nemingxuma yazo ngamanzi anebhlitshi. Hlamba amalaphu, izponji neebhrashi zokuhlamba izitya ngebhlitshi, iziponji zitshintshe emva kweeveki ezimbalwa.
n Ungashiyi ukutya apho kuza kubiza amaphela, iimpukane neempuku kuba zisasaza iintsholongwane.
n Ukutya kuthenge kukwindlela ekufanele kugcinwe ngayo. Ukutya okugcinwa efrijini makubande ukanti okugcinwa kwisikhenkcisi makube ngumkhenkce. Wujonge umhla wokuphelelwa.
n Khupha umkhenkce ekutyeni ukubeke efrijini. Kuthabatha ixesha elide ukukhupha umkhenkce kodwa kukhuselekile.
n Yipheke ivuthwe inyama yenkuku ukuze kufe iintsholongwane. Nenyama yehagu nenyama egrayiweyo kuba zingaba nentsholongwane eyingozi xa zingavuthisiswanga.
Ungaze...
n Ushiye ukutya apho kushushu ixesha elingaphezulu kweyure. Ukutya akufanelanga kutyiwe bukubushushu basendlwini. Ukutya okubandayo kufuneka kubande, ukanti okushushu kufuneka kubeshushu. Khumbula ukutya konakala kungaphelanga neyure kwindawo eshushu kuze kungakhuseleki, nkqu nokuba kuselangeni.
n Ucinge ukuba ukufaka ukutya efrijini kubulala iintsholongwane. Xa sekonakele konakele ke. Kunjalo nakwizinongo. Ukutya okunezinongo ezininzi, nako kuyonakala qha akukhawulezi ukuba nevumba!
n Ufake amachiza okucoca njengebhlitshi nesicoci segumbi langasese kwipakethe enye nokutya.
n Uthenge iinkonkxa zokutya ezinerusi okanye eziboqokileyo. Iinkonkxa ezikhangeleka "zimdaka" ngebala okanye ziqumbile zingakugulisa.
n Ushiye ukutya okonakalayo emotweni eshushu.
n Usebenzise ipleyiti okanye izitya ebezinenyama ekrwada zingahlanjwanga.
Unyango
Ukuhlanza norhudo yindlela yendalo umzimba okhupha ngayo ukutya okonakeleyo kodwa ungakunyanzeli ukuhlanza.
Kwakuphuma oko kutya uqala ukuziva ubhetele. Thintela ukuphela kwamanzi emzimbeni ngokusela izinto ezingamanzi.
Buyisela amanzi emzibeni ngokusela ilitha yamanzi abilisiweyo apholiswa agalelwa itispuni yetyuwa necephe leswekile.
Fuman' unyango xa unotyatyazo ngaphezulu kweeyure ezingama-24, okanye xa uhlanze ngaphezu kweeyure ezili-12. [x]
